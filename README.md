# corona-rhein-neckar

Small scraper, which displays some COVID-19 statistics of the Rhein-Neckar-Kreis, Germany.


## Install and run

1. Install the dependencies and run it

   ```
   npm install
   node index.js
   ```

2. Browse to http://localhost:3000
